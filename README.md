## Project Overview

This project is a Laravel backend for a web application. The frontend is built with Vue.js and is located in a separate repository. The setup and deployment of this project are managed by Ansible, which is available in the following repository: [qfranklin/ansible](https://github.com/qfranklin/ansible).

## Features

- User registration with optional name, required email, and required password.
- Google login integration.
- API endpoints for frontend interaction.

## Project Structure

- **Backend**: Laravel (this repository)
- **Frontend**: Vue.js (separate repository)
- **Deployment**: Managed by Ansible ([qfranklin/ansible](https://github.com/qfranklin/ansible))
